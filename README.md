# 🕹️ Pacman Python Edition

A faithful recreation of the classic arcade masterpiece. This project is built using **Python 3** and the **Pygame** library, showcasing sprite management, collision detection, and game loop logic.

---

## ✨ Features
* **Classic Gameplay:** Navigate through the maze and avoid obstacles.
* **Sprite Animations:** Dynamic movement logic for Pacman.
* **Custom Level Grid:** Easy-to-edit level maps based on a coordinate system.
* **Physics Engine:** Custom-built collision detection with walls and boundaries.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** Pygame
* **Graphics:** Custom PNG sprites

---

## 🚀 How to Play

### Prerequisites
Make sure you have Python installed on your system. You will also need the `pygame` library.

### 1. Install Pygame
Open your terminal and run:
```bash
pip install pygame

```

2. Clone & Run

# Clone the repository
```bash
git clone [https://github.com/Maxcer222/pacman.git](https://github.com/Maxcer222/pacman.git)
```
# Enter the directory
```bash
cd pacman
```
# Run the game
```bash
python Pacman.py
```
